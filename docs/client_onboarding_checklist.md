# TorqueLeads Client Onboarding Checklist

## Before the Call
- [ ] Pull fleet data for their area (FMCSA + Google Maps)
- [ ] Check their website/Facebook
- [ ] Note their Google reviews
- [ ] Prepare example ads

---

## On the Call — Info to Collect

### Business Info
- [ ] Shop name
- [ ] Owner name
- [ ] Phone number
- [ ] Email
- [ ] Physical address
- [ ] Website URL
- [ ] Years in business

### Shop Details
- [ ] Number of bays
- [ ] Number of techs
- [ ] Specialties (fleet, retail, both)
- [ ] Average ticket size
- [ ] Current monthly revenue (if they share)

### Marketing History
- [ ] What have they tried before?
- [ ] What worked / didn't work?
- [ ] Current lead sources

### Goals
- [ ] Target number of new fleet accounts
- [ ] Growth goals (revenue, bays, techs)
- [ ] Timeline

---

## After They Sign Up

### Day 1: Setup
- [ ] Add to GHL CRM
- [ ] Create their ad account (or get access to existing)
- [ ] Install Meta Pixel on their website
- [ ] Set up conversion tracking

### Day 1-2: Assets
- [ ] Get shop logo (high-res PNG)
- [ ] Get 5-10 shop photos (bays, trucks, team)
- [ ] Get any existing videos
- [ ] Write ad copy (3 variations minimum)

### Day 2-3: Campaign Build
- [ ] Create Facebook Business Page (if needed)
- [ ] Connect Instagram account
- [ ] Build custom audience (fleet companies in area)
- [ ] Create lookalike audience
- [ ] Set up campaign structure
- [ ] Create ad sets (by audience)
- [ ] Upload creatives
- [ ] Set budget

### Day 3-4: Launch
- [ ] Review everything with client (optional)
- [ ] Launch campaign
- [ ] Monitor first 24 hours closely
- [ ] Adjust if needed

### Ongoing
- [ ] Weekly check-in (first month)
- [ ] Monthly report
- [ ] Optimization every 2 weeks
- [ ] Creative refresh every 30 days

---

## Assets Needed from Client

| Asset | Format | Notes |
|-------|--------|-------|
| Logo | PNG, high-res | Transparent background preferred |
| Shop photos | JPG/PNG | Bays, trucks, team, exterior |
| Videos | MP4 | Optional but great for Reels |
| Testimonials | Text/video | Customer quotes |
| Before/after | Photos | Great for ads |

---

## GHL Setup

- [ ] Create contact
- [ ] Add to pipeline (New Lead stage)
- [ ] Set up SMS automation
- [ ] Configure lead notifications
- [ ] Test the flow

---

## Payment Setup

- [ ] Send Stripe payment link
- [ ] Confirm payment received
- [ ] Set up recurring billing
- [ ] Send receipt/invoice
