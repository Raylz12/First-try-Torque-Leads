# TorqueLeads Client Onboarding Checklist

## Before the Call
- [ ] Pull fleet data for their area (FMCSA + Google Maps)
- [ ] Check their website/Facebook
- [ ] Note their Google reviews
- [ ] Prepare example ads

## On the Call
- [ ] Confirm shop details (name, location, bays)
- [ ] Get average ticket size
- [ ] Understand current marketing efforts
- [ ] Set expectations (timeline, results)
- [ ] Discuss pricing and close

## After Close — Collect from Client
- [ ] Shop logo (high res PNG/SVG)
- [ ] 5-10 shop photos (bays, trucks, team)
- [ ] Facebook Page access (admin or partner)
- [ ] Instagram account access
- [ ] Business address (for targeting)
- [ ] Phone number for ads
- [ ] Website URL
- [ ] Any existing testimonials/reviews

## Setup in GHL
- [ ] Create contact in GHL
- [ ] Add to "Won" stage in pipeline
- [ ] Set up SMS workflow trigger
- [ ] Create client folder in Drive

## Setup in Meta
- [ ] Create ad account (if needed)
- [ ] Install Pixel on their site (if they have one)
- [ ] Create custom audience for their area
- [ ] Build first ad set
- [ ] Create 3-5 ad variations
- [ ] Set budget and schedule
- [ ] Submit for review

## Launch
- [ ] Ads approved and running
- [ ] Confirm tracking is working
- [ ] Send "you're live" message to client
- [ ] Schedule first check-in (1 week)

## Ongoing
- [ ] Weekly performance check
- [ ] Optimize underperformers
- [ ] Rotate creative every 2 weeks
- [ ] Monthly report to client
