# TorqueLeads VSL — Loom Recording Script
**Target length:** 8–9 minutes  
**Format:** Rhett on camera (or voice over slides), conversational tone  
**Goal:** Pre-sell the call. They should show up already wanting to work with you.  
**Rule:** DO NOT mention pricing. The call is where that happens.

---

## SETUP NOTES BEFORE YOU RECORD
- Record in a shop environment if possible — or at a clean desk with something diesel-related visible
- Speak like you're talking to one shop owner, not a crowd
- Slight pauses after big statements are good — let it land
- Don't rush. Calm confidence is more persuasive than high energy
- You can do this in one take. It doesn't need to be perfect — it needs to be real

- [ ] Look at the camera, not the screen
- [ ] Keep water nearby
- [ ] Record in one take if possible — the realness beats a polished retake
- [ ] Upload to YouTube (unlisted) or Loom, then send Queefus the link to embed
